Real-time phishing detection system using hybrid machine learning models to analyze email content and URLs, with a browser-based approach for proactive user security.
# Real-Time Phishing Detection using Machine Learning

## Overview
This project focuses on detecting phishing attacks in real-time using a hybrid machine learning approach. It aims to improve email security by analyzing both email content and URLs.

## Features
- Real-time phishing detection
- Hybrid ML model combining text and URL analysis
- Browser-based detection approach (extension concept)
- Explainable risk scoring for better user awareness

## Machine Learning Approach
- Text Analysis:
  - TF-IDF Vectorization
  - Logistic Regression

- URL Analysis:
  - Lexical feature extraction
  - Machine Learning classification

- Hybrid Model:
  - Combines both models for improved accuracy

## Technologies Used
- Python
- Machine Learning (Scikit-learn)
- Flask (for backend API - planned)
- Chrome Extension (planned)

## How It Works
- Extracts email content and URLs
- Processes data using ML models
- Calculates risk score
- Displays warning to users in real-time
